# AI Event Concierge Platform

A full-stack AI-powered platform that helps users plan corporate offsite events.

## Features
- Natural language event planning
- AI-generated venue recommendations
- Real-time results display
- Search history stored in database

## Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: Django + Django REST Framework  
AI: Google Gemini API  
Deployment: Vercel (frontend), Render (backend)

## Live Demo
https://ai-event-concierge-frontend.vercel.app

## Backend API
https://ai-event-concierge-backend.onrender.com

## Run Locally

1. Clone repo
2. Create virtual environment
3. Install dependencies
4. Add `.env`
5. Run server
